OVERVIEW OF THE PROJECT CONTACT BOOK:

Contact Book is a simple Python program that helps you to save, search, update, and delete the contact details of people. It is a digital version of traditional phonebook which let the user efficiently manage contact information. It's a good?project that help beginners to learn concepts like file handling, data structures, functions and CRUD operations (Create, Read, Update, Delete) in python.

A typical Contact Book includes:
